USER INSTRUCTIONS:

Es una adaptación del proyecto del capítulo 1. Hay dos versiones del proyecto: versión Eclipse y 
versión BlueJ.

El proyecto es un proyecto muy simple para mostrar algunas características de los objetos.

El proyecto se debe usar después de haber trabajado con el proyecto T01.1-Figuras. Añade
una interfaz y una clase nueva (Casa y CasaImpl) al paquete fp.figuras para dibujar una casa.
Se puede experimentar modificando el código fuente.
